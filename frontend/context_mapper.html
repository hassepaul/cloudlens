ℹ How context mapping works

CloudLens scans all cost records from the past 30 days and applies a three-tier attribution heuristic to each resource:

  1. Explicit tags — Checks for product, app, application, service, component tags.
  2. K8s namespace — Extracts namespace from resource name or annotation (namespaces/checkout, ns:payments etc.).
  3. Resource name inference — Takes the first meaningful token of the resource name, filtered against common generic prefixes.

Feature attribution uses feature, feature_flag, experiment, variant, epic tags. Team attribution uses team, owner, squad, tribe, cost_center tags. Coverage improves naturally as teams adopt tagging conventions.
